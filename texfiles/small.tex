\begin{list1}
\item Step 1: install \TeX Live \link{https://www.tug.org/texlive/}\\
 or other \LaTeX, and Latexmk \link{https://mg.readthedocs.io/latexmk.html}
\item Step 2: git clone \link{https://codeberg.org/kramse/security-courses}
\item Step 3: Make sure TEXINPUTS can find the texfiles, add to .bashrc or .profile:\\
\verb+export TEXINPUTS=/home/$USER/projects/security-courses//:+
\item Step 4: Update your \verb+~/.latexmkrc+
\vskip 5mm
\item Finally Go to \verb+security-courses/texfiles+ and run \verb+latexmk small.tex+
\item another option is to use \link{https://www.overleaf.com/}\\ The easy to use, online, collaborative \LaTeX\ editor
\end{list1}
